naive-ui-editor-view文档 ​

介绍 ​

naive-ui-editor-view是基于vue-dompurify-html封装的一个富文本内容渲染组件

安装 ​

bash
pnpm add naive-ui-editor-view vue-dompurify-html
1

也可以使用npm、yarn等安装。
